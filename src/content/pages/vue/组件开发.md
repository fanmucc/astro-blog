# Vue 组件开发

组件是 Vue.js 最强大的功能之一。

## 组件基础

组件是可复用的 Vue 实例，它们有自己的数据、方法和生命周期。

```vue
<template>
	<div class="hello">
		<h1>{{ msg }}</h1>
	</div>
</template>

<script>
export default {
	name: "HelloWorld",
	props: {
		msg: String,
	},
};
</script>
```

这也是一个一级页面，没有子菜单。
